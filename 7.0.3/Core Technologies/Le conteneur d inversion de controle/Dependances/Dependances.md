# _**Dépendances**_

Un application d'entreprise type ne consiste pas en un unique objet (ou  **bean** dans le langage _Spring_). Même l'application la plus simple posséde plusieurs objets travaillant ensemble pour présenter à l'utilisateur final une application cohérente. La section suivante explique comment définir un certain nombre de **beans** qui sont autonomes à une application entièrement réalisée où les objets coopérents pour atteintre un but.
